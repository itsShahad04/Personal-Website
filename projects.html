Technical Projects

Student Planner

The student planner is a comprehensive platform to centralize academic management, replacing fragmented tools with a unified system for task tracking, GPA calculation, and automated study scheduling.

Technical Implementation & QA:
  • Requirement Engineering (RE): Led the transition from initial elicitation to formal documentation. Authored a structured SRS and modeled system behavior through Use Case and Context Diagrams.

  • Multi-Tier Testing: Developed a rigorous plan including Unit Testing (Authentication, GPA calculation), Component Testing (Task-Notification sync), and System Testing for performance under load.

  • User Acceptance Testing (UAT): Conducted scenarios focused on real-world student workflows, such as exam preparation via the Study Planner.

  • Software Architecture: Designed a secure architecture utilizing password hashing, JWT-based session management, and 99% uptime targets.
Core Features:
  • Intelligent Study Planning: Algorithmic generation of personalized study slots based on availability.
  • Academic Tracking: Automated GPA calculation with performance visualization.
  • Dynamic Task Management: Full CRUD system with automated 24-hour Firebase reminders.
  • UI/UX Design: Prioritized WCAG 2.1 accessibility and a "three-click" navigation rule.
Library Management System

The Library Management System is designed and developed to minimize manual workloads while maintaining a consistent workflow and an intuitive interface. This project evolved over two semesters, integrating foundational knowledge from three core courses:

  • Introduction to Database Management: Designed the system architecture and page hierarchy using UML diagrams, and implemented the backend structure using MySQL.

  • Introduction to Web Development: Built the frontend using HTML and CSS, and utilised JSP to connect the application to a database and an Apache server for real-time interactive testing.

  • Software Interface Design: Applied UI/UX principles to ensure a cohesive, accessible, and user-friendly experience across the entire platform.
Core Features:
  • Catalog & Inventory Management:Digital repository for book metadata and automated stock tracking
  • Circulation & User Management Streamlined borrowing/returning interface with due date notifications
  • Technical Architecture:Relational database integration (MySQL) for high data integrity
  • UI/UX & Accessibility: Intuitive navigation and inclusive design adhering to software interface principles
